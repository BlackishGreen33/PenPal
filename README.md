- 🎉 init：项目初始化

- 🐞 fix：修复bug
- 📃 docs：文档修改
- 🌈 style：代码样式修改，不影响原代码逻辑

### 💻 本地调试

记得先配置环境变量：

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_CLERK_SIGN_IN_URL=
NEXT_PUBLIC_CLERK_SIGN_UP_URL=

LIVEBLOCKS_SECRET_KEY=

SENTRY_AUTH_TOKEN=
```

接着拉取代码和安装依赖：

```bash
$ git clone https://github.com/BlackishGreen33/PenPal.git
$ cd PenPal
$ bun install
$ bun run dev
```
